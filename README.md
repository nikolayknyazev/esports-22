# Cyber Sports Portal

**Cyber Sports Portal** — киберспортивный портал для просмотра игроков, команд, турниров и рейтингов.

Проект разрабатывается командой **«22:00»**.

## Идея продукта

Портал помогает быстро находить структурированную информацию об игроках, командах, турнирах и рейтингах.
