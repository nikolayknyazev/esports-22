## Что отражено в карте

1. Система ролей: администратор, зритель и аналитик.
2. Структура User Story Map: активности, пользовательские шаги и этапы релизов.
3. Приоритизация функциональности по релизам: MVP, второй релиз и третий релиз.
4. Основные функциональные блоки портала:
   - просмотр игроков;
   - просмотр команд;
   - просмотр турниров;
   - расчет рейтинга.
