# Задание 1: Бизнес- и системный анализ — User Story Map

## Описание

В рамках задания разработан User Story Map для киберспортивного портала "arena-22".

### Ссылка на карту: 

https://unidraw.io/app/board/6a1358b6363004119d2a

### Данная структура обеспечивает следующее:

1. Систему ролей: администратор, зритель и аналитик
2. Трехуровневую модель вида: активности, пользовательские шаги как историй и этапы релизов
3. Приоритеты создания функциональностей в виде трех этапов релиза: первого (MVP), второго (2.0) и третьего (3.0)
4. Основные бизнес требования
   1) Просмотр игроков
   2) Просмотр команд
   3) Просмотр турниров
   4) Расчет рейтинга
