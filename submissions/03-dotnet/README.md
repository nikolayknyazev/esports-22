# Задание 3: .NET

## Описание

В рамках задания были разработаны методы бизнес-логики для портала киберспортивных турниров CyberSportsPortal на языке C# (.NET 8).

## Что реализовано

1. Статус турнира (GetTournamentStatus) - определение состояния турнира («Не начался» / «В процессе» / «Окончен») на основе сравнения текущей даты с полями StartDate и EndDate.

2. Доход команды за год (GetTeamIncomeForYear) - расчёт суммарного призового дохода команды за указанный календарный год. Для доступа к данным о турнирах и призовых фондах сервис был доработан внедрением зависимости CyberSportsContext через DI, что позволило напрямую работать с сущностью Team и её навигационными свойствами. Алгоритм фильтрует результаты команды по году окончания турнира, сопоставляет занятое место (Place) с распределением призового фонда (TournamentPrize) и суммирует выплаты.

3. Количество игроков из страны (GetPlayersFromCountryCount) - подсчёт числа игроков указанной страны с использованием LINQ-метода Count с предикатом.

4. Список команд-участников (GetTeamParticipantsNameString) - формирование строки с названиями команд через запятую без лишней запятой в конце с использованием string.Join.

5. Фильтрация игроков (FilterPlayers) - поиск игроков по вхождению подстроки в никнейм или полное имя (имя + фамилия) с регистронезависимым сравнением.

6. Сортировка таблицы призов (ComparePrizes) - компаратор для сортировки призовых сумм, реализованный через int.TryParse и IComparable.CompareTo, возвращающий значения -1 / 0 / 1 по договорённости интерфейса компаратора.

7. Расчёт вероятностей показа рекламы (GetProbabilities) - вычисление целочисленных вероятностей (от 1 до 100) показа рекламы для каждого рекламодателя пропорционально его платежам за текущий год. Реализована обработка краевых случаев: нулевая общая сумма платежей и правило "не меньше 1" для каждой вероятности. Округление вниз выполняется через Math.Floor после умножения доли на 100.
